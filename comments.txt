>git init [to initialize git]

>git status [to list all untracked files]

>git add -A [to adding untracked files to staging area]

>git commit -m "commit message" [to commit added file to local repository]

>git clone [to create exact copy from remote repository to local workspace]

>git pull [to take changes from remote repository to local workspace]

>git push [to send changes from local repository to remote repository]

>git config --list [list configuration]

>git config --global user.name "username" [to add username]

>git config --global user.email "useremail" [to add useremail]
